# ⚔️ Card Battle TCG

ポケモンカードゲーム風のオリジナル・トレーディングカードゲーム（TCG）の**ブラウザ対戦アプリ**。
ビルド不要のバニラ JavaScript（ES Modules）で動作し、シンプルな AI と 1 人で対戦できます。

> 本物のポケモンの名称・イラスト・カードデータは使用していません。
> すべてオリジナル名＋汎用エレメント（炎/水/草/雷/超/闘/無）のプレースホルダーです。

## 遊ぶ

GitHub Pages: <https://tr-hirama.github.io/card-battle-tcg/>

ローカルで動かす場合は、ES Modules のため **HTTP 配信**が必要です（`file://` では動きません）。

```sh
# 任意の静的サーバでルートを配信して index.html を開く
npx serve .
# または Python: python -m http.server
```

## ルール（v1）

- デッキ 60 枚・初期手札 7 枚・サイド 6 枚・マリガンあり
- 先攻・後攻はランダム。**先攻の最初のターンはドロー・ワザ・サポート不可**
- 毎ターン：1 ドロー → ベンチ展開／進化／エネルギー 1 枚／トレーナー／にげる → **ワザ（使うとターン終了）**
- 弱点 ×2・抵抗力 −30
- 特殊状態：どく／やけど／ねむり／マヒ／こんらん（ターン間にチェック）
- 相手をきぜつさせるとサイドを 1 枚取る。**サイド 6 枚を取りきれば勝ち**（ほかに山札切れ・場のポケモン全滅でも決着）

## 構成

| ファイル | 役割 |
|---|---|
| `js/data/cards.js` | カードマスタ＋デッキ定義 |
| `js/engine/game.js` | ゲーム本体（ルール・状態管理） |
| `js/engine/effects.js` | ワザ・トレーナー効果の解釈（拡張ポイント） |
| `js/ai/ai.js` | ルールベース AI |
| `js/ui/ui.js` | 盤面・手札・ログの描画 |
| `js/main.js` | 操作とゲーム進行の配線（コントローラ） |
